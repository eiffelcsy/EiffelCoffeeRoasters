# Origin artwork

(paths are set per lot in `src/data.js`):

- `guji-bule-hora.jpg`
- `excelso.jpg`

Portrait 4:5 works best (e.g. 1200×1500). Until a file exists, the site shows
an abstract gradient placeholder built from that coffee's flavour palette.
